# Running locally

## 1. Clone repo

```
git clone https://github.com/charlenelbb/dashboard-full-stack.git
```

## 2. Install PostgresQL

Download it from here: https://www.postgresql.org/download/

## 3. Run Backend

```
cd dashboard-full-stack
yarn
yarn server
```

## 4. Run Frontend

```
# open new terminal
yarn client
```
